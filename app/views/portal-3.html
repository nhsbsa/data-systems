Data services

Reports and data to help NHS customers track trends, inform decisions and support policy

Here are the systems you have access to:

ePACT2 System

Use this service to access prescription data.

The ePACT2 service will allow you to access interactive and easy to use reports and dashboards, view high level data summaries down to individual prescription items and patient level detail, build bespoke analysis and export data.

Access ePACT2 Data
eDEN System

Use this system to access dental data.

eDEN is an online application designed to streamline dental data management. With eDEN, users can swiftly access dental information, ensuring a faster and more efficient retrieval process.

The platform offers user-friendly reporting tools, facilitating easy management of reports. Additionally, eDEN enables users to refresh and export dashboards and flat data tables, enhancing the overall data handling experience. The application also provides visually engaging data visualizations, offering a comprehensive and intuitive overview of dental data.

Access eDEN Data
eOPS System

Use this system to access ophthalmic data.

eOPS is an online application designed to optimize the handling of ophthalmic data. Users benefit from access to vital information, ensuring a swift and efficient data retrieval process.

The platform offers intuitive and user-friendly reporting tools, empowering users to easily manage reports with enhanced efficiency. Additionally, eOPS facilitates the seamless refresh and exportation of dashboards and flat data tables, further streamlining data manipulation. The application enhances data comprehension through visually appealing data visualizations, providing a comprehensive and insightful overview of ophthalmic data.

Access eOPS Data
